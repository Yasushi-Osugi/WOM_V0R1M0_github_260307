------------------------------------------------------------
Architectural rules (must follow)
------------------------------------------------------------

1. Flow/Event is the source of truth.

2. StateView is a derived view generated from FlowEvents.

3. StateView must never be directly mutated.

4. Operators must modify planning objects (FlowEvents or Lots).

5. Deterministic ordering must be preserved.

Use explicit ordering keys such as:

time_bucket
event_priority
creation_sequence

6. No randomness.

7. No external dependencies.

Pure Python only.
